# Smart Calculator 

A Python-based calculator that supports multiple number operations with advanced features.

## Features 
- Multi-number operations (Add, Subtract, Multiply, Divide)
- History tracting
- Duplicate detection
- Clean input system

## Tech Used
- Python

## How to run
...bash

python smart_calculator.py
